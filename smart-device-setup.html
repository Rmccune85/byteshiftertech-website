Smart Device Setup

Connection and basic setup for supported smart televisions, streaming devices, cameras, speakers, and connected home devices.

Request Service

What This Service Includes

Compatibility Review

The device, wireless network, required application, account requirements, and available setup methods are reviewed.

Network Connection

The device is connected to the available Wi-Fi or supported Ethernet network.

Application Setup

The required manufacturer application is installed and configured on one supported phone or tablet.

Account Connection

The device is connected to the customer’s existing account or a new account is created when requested.

Basic Configuration

Device names, basic settings, notifications, and supported preferences are configured.

Final Testing

Network connection, application control, account access, and basic device operation are tested.

Supported Device Setup

Smart Televisions

Wi-Fi connection, system updates, streaming application setup, account sign-in, and basic picture or sound settings.

Streaming Devices

Roku, Fire TV, Apple TV, Chromecast, and other supported streaming device setup.

Smart Speakers

Supported smart speakers and displays can be connected, updated, named, and linked to the required application.

Security Cameras

Supported indoor or outdoor cameras can be connected to Wi-Fi and configured in the manufacturer application.

Video Doorbells

Supported video doorbells can be connected to the network and configured after physical installation is complete.

Other Connected Devices

Supported plugs, lights, thermostats, and other common smart devices may also be configured.

Common Problems We Can Help With

Device Will Not Connect to Wi-Fi

The device cannot find, join, or remain connected to the wireless network.

Application Cannot Find the Device

The required mobile application cannot detect, pair with, or complete setup for the device.

Account or Sign-In Problems

The device requires an account, password reset, verification code, or subscription sign-in.

Streaming Applications Not Working

Streaming applications need to be installed, updated, activated, or signed in.

Device Shows Offline

A previously working device appears disconnected, unavailable, or offline in its application.

New Device Setup

A newly purchased device needs to be connected, updated, configured, and tested.
